Repaso Prioritario - Clase 1

Versión corta para una primera clase. Preguntás primero y después muestran la respuesta.

1. Fundamentos Web
¿Qué es Full Stack?
Perfil que comprende varias capas de una aplicación: frontend, backend y base de datos, entendiendo cómo se integran.
¿Qué es el frontend?
Es la parte visual de la aplicación, la que ve e interactúa el usuario.
¿Qué es el backend?
Es la capa que procesa la lógica, aplica reglas y responde pedidos del frontend.
¿Qué es una base de datos?
Es el sistema donde se guarda la información de forma persistente.
¿Qué es HTTP?
Es el protocolo de comunicación entre cliente y servidor en la web.
¿Qué es JSON?
Es un formato de intercambio de datos muy usado entre frontend y backend.
2. HTML
¿Qué es HTML?
Es un lenguaje de marcado que define la estructura de una página web.
¿Qué hace <!DOCTYPE html>?
Indica al navegador que el documento está escrito en HTML5.
¿Qué es <head>?
Contiene configuración no visible, como título, metadatos, CSS y scripts.
¿Qué es <body>?
Contiene todo lo visible para el usuario dentro de la página.
¿Qué es <meta>?
Es una etiqueta que da información sobre la página, pero no muestra contenido visible.
¿Qué es una etiqueta HTML?
Es una marca que define un elemento del documento, como un párrafo, título o formulario.
¿Qué es un atributo?
Es información adicional dentro de una etiqueta, como id, class, src o href.
¿Qué hace <a>?
Crea enlaces y usa href para indicar el destino.
¿Qué es <form>?
Es el contenedor que agrupa campos para ingresar y enviar datos.
¿Qué es <input>?
Es un campo de entrada que permite capturar datos del usuario.
3. CSS
¿Qué es CSS?
Es el lenguaje que define el aspecto visual de un documento HTML.
¿Qué es un selector CSS?
Es la parte de la regla que indica a qué elemento se le aplicará el estilo.
¿Cómo se vincula una hoja de estilos externa?
Con la etiqueta <link> dentro del <head>.
CSS en línea, interno y externo
En línea va en el atributo style, interno dentro de <style> y externo en un archivo .css.
4. JavaScript y Node.js
¿Qué es JavaScript?
Es un lenguaje de programación que agrega lógica e interactividad a la web.
¿Qué es Node.js?
Es un entorno que permite ejecutar JavaScript fuera del navegador.
¿Qué es asincronía?
Es la capacidad de ejecutar tareas lentas sin bloquear el hilo principal.
¿Qué es el Event Loop?
Es el mecanismo que coordina la ejecución de tareas pendientes cuando el hilo principal queda libre.
¿Cómo se vincula JavaScript al HTML?
Con la etiqueta <script>, ya sea escribiendo código dentro o usando src para un archivo externo.
5. Programación
¿Qué es una variable?
Es un espacio de memoria con nombre donde se almacena un valor.
¿Qué es una función?
Es un bloque de código reutilizable que realiza una tarea específica.
¿Qué es un parámetro?
Es la variable declarada en la definición de una función para recibir datos.
¿Qué es un argumento?
Es el valor real que se le pasa a una función cuando se la llama.
¿Qué es un objeto?
Es una estructura que agrupa datos relacionados, normalmente en pares clave-valor.
¿Qué paradigma usa JavaScript?
JavaScript es multiparadigma: permite programar de forma funcional, orientada a objetos e imperativa.
