Creá un archivo index.html con la estructura básica de una página web (<!DOCTYPE html>, html, head, body).
Dentro del body, agregá un título principal con el nombre de la página usando h1.
Agregá un párrafo descriptivo explicando de qué trata la página.
Insertá un botón con el texto Haz clic aquí.
Creá un archivo styles.css y vinculalo al HTML con la etiqueta link.
Aplicá estilos generales al body: cambiar color de fondo, centrar el contenido y elegir una fuente.
Dale estilo al título: cambiar color, tamaño de letra y separación.
Diseñá el botón: color de fondo, borde redondeado, padding, cursor pointer y efecto hover.
Creá un archivo script.js y vinculalo al HTML al final del body.
Desde JavaScript, capturá el botón usando document.querySelector() o getElementById().
Agregale un evento click al botón.
Cuando el usuario haga clic, mostrar un mensaje en pantalla con alert() o cambiar el texto del párrafo.
Como mejora extra, hacer que al presionar el botón también cambie el color de fondo de la página.
Probá todo en el navegador y verificá que HTML estructura, CSS diseña y JavaScript agrega interacción.
