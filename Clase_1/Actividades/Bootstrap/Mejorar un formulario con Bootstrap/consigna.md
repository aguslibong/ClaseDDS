# Ejercicio: Mejorar un formulario con Bootstrap

## Objetivo
Aprender a transformar un formulario HTML común en un formulario moderno utilizando clases de Bootstrap.

## Consigna
Se entrega un formulario HTML básico sin estilos.

Debés mejorarlo aplicando Bootstrap para que tenga una apariencia profesional.

## Debés utilizar al menos:

- `form-control`
- `btn`
- `btn-primary`
- `container`
- márgenes o espacios (`mb-3`, `mt-5`, etc.)

## Pasos sugeridos

### Parte 1 - Preparar Bootstrap
1. Vincular Bootstrap mediante CDN o local.
2. Verificar que la página cargue correctamente.

### Parte 2 - Mejorar estructura
1. Encerrar todo en un `.container`.
2. Agregar un título.

### Parte 3 - Mejorar inputs
1. Aplicar `form-control` a:
   - nombre
   - email
   - mensaje

### Parte 4 - Mejorar botón
1. Reemplazar botón común por clases Bootstrap.

### Parte 5 - Probar resultado
1. Guardar cambios.
2. Abrir navegador.
3. Comparar antes y después.

## Qué observar
- Inputs más grandes y prolijos.
- Espaciado automático.
- Botón moderno.
- Mejor alineación general.

## Resultado esperado
Un formulario limpio, usable y profesional sin escribir CSS propio.

## Conclusión
Bootstrap permite mejorar interfaces rápidamente usando clases ya preparadas.

En vez de diseñar inputs y botones desde cero, reutilizamos componentes estándar y consistentes.
