# Desafío: Primeros pasos con Bootstrap

## Objetivo
Aprender a incorporar Bootstrap en un proyecto HTML, utilizar su documentación oficial y aplicar componentes prediseñados.

## Consigna
Debés crear una página HTML simple utilizando Bootstrap.

El desafío consiste en:

1. Vincular Bootstrap al proyecto.
2. Crear una página HTML funcional.
3. Ingresar a la documentación oficial de Bootstrap.
4. Copiar un componente desde la web oficial.
5. Integrarlo correctamente en tu página.
6. Personalizar al menos un texto del componente.

## Podés elegir una de estas dos formas para incorporar Bootstrap:

### Opción A: CDN
Vincular Bootstrap desde internet usando `<link>` y `<script>`.

### Opción B: Local
Descargar Bootstrap y enlazar los archivos desde tu proyecto.

## Pasos sugeridos

### Parte 1 - Crear proyecto
1. Crear carpeta del proyecto.
2. Crear archivo `index.html`.
3. Crear estructura HTML básica.

### Parte 2 - Incorporar Bootstrap
Elegí una opción:

#### CDN
1. Ingresá a https://getbootstrap.com/
2. Buscá "Get Started".
3. Copiá el `<link>` CSS.
4. Pegalo dentro del `<head>`.
5. Copiá el script JS.
6. Pegalo antes de cerrar `</body>`.

#### Local
1. Descargá Bootstrap.
2. Guardá la carpeta css/js dentro del proyecto.
3. Enlazá los archivos locales en el HTML.

### Parte 3 - Usar documentación oficial
1. Entrá a la sección Components de Bootstrap.
2. Elegí uno de estos elementos:

- Navbar
- Card
- Alert
- Button
- Modal
- Carousel

3. Copiá el código HTML del componente.
4. Pegalo en tu página.
5. Cambiá textos, títulos o colores.

### Parte 4 - Mejorar página
Agregar:

- un título principal
- un contenedor `.container`
- margen o padding usando clases Bootstrap

## Qué observar
- Sin Bootstrap la página se ve simple.
- Con Bootstrap aparecen estilos automáticamente.
- No fue necesario escribir CSS propio.

## Resultado esperado
Una página HTML moderna usando componentes reales de Bootstrap.

## Conclusión
Bootstrap acelera el desarrollo porque trae estilos, grid responsive y componentes ya diseñados.

En vez de crear todo desde cero con CSS, reutilizamos soluciones probadas y profesionales.
