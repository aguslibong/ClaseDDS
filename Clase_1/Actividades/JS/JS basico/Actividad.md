# 1 
crear una funcion que le llega por parametro un array de 50 numeros. A partir de ese array resolvé los siguientes ejercicios: 
1. recorré el array completo y mostrá por consola la suma total de todos los números. 
2. Segundo, obtené y mostrá cuál es el número mayor y cuál es el número menor dentro del array. 
3. Tercero, generá un nuevo array que contenga solamente los números pares del array original y mostralo por consola.
