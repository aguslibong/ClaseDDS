# Ejercicio: Atributos HTML5 en formularios

## Objetivo
Aplicar atributos modernos de HTML5 para mejorar la experiencia del usuario y la validación básica de formularios.

## Consigna
Se entrega un formulario incompleto.

Debés completarlo agregando correctamente los siguientes atributos HTML5:

- `required`
- `placeholder`
- `autofocus`

## Reglas
- Todos los campos deben ser obligatorios.
- El cursor debe posicionarse automáticamente en el primer campo al cargar la página.
- Cada input debe mostrar un texto de ayuda interno.

## Importante
No usar JavaScript.
No agregar CSS.

## Resultado esperado
Un formulario más profesional, cómodo de usar y con validaciones básicas nativas del navegador.
