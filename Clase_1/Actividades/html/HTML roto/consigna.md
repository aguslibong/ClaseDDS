# Ejercicio: HTML roto

## Objetivo
Detectar y corregir errores básicos en la estructura de un documento HTML.

## Consigna
A continuación se entrega un archivo HTML con varios errores de escritura y de estructura.  
El objetivo es **arreglar el código para que el documento quede correctamente armado** y pueda visualizarse bien en el navegador.

## Tenés que revisar especialmente:
- etiquetas de apertura y cierre
- etiquetas mal anidadas
- estructura básica de un documento HTML
- texto que debería quedar dentro de la etiqueta correcta

## Instrucciones
1. Abrí el archivo entregado.
2. Identificá los errores.
3. Corregí el HTML sin cambiar el contenido textual.
4. Probá el resultado en el navegador.

## Condición
No hace falta agregar estilos ni JavaScript.  
Solo hay que dejar el HTML correctamente estructurado.

## Resultado esperado
La página debe mostrar:
- un título principal
- un párrafo de bienvenida
- una lista con 3 elementos
- un enlace
- un pie de página
