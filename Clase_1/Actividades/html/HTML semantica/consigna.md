# Ejercicio: HTML semántico

## Objetivo
Comprender la diferencia entre usar etiquetas genéricas (`div`) y etiquetas semánticas de HTML5.

## Consigna
Se entrega una página armada únicamente con etiquetas `div`.

Tu trabajo consiste en **reestructurar el documento reemplazando los `div` principales por etiquetas semánticas**, manteniendo el mismo contenido.

## Debés utilizar:
- `header`
- `nav`
- `main`
- `section`
- `footer`

## Importante
- No cambiar textos.
- No agregar estilos.
- No usar JavaScript.
- Solo mejorar la estructura HTML.

## Resultado esperado
El código debe quedar más claro, ordenado y profesional.
