# Ejercicio: Formulario de inscripción

## Objetivo
Aprender a construir formularios HTML utilizando etiquetas correctas y campos adecuados según el tipo de dato.

## Consigna
Desarrollar una página HTML que contenga un formulario de inscripción.

El formulario debe solicitar los siguientes datos:

- Nombre completo
- Correo electrónico
- Edad

Además, debe incluir un botón para enviar la información.

## Requisitos técnicos
- Utilizar la etiqueta `<form>`
- Usar `<label>` en cada campo
- Asociar correctamente cada label con su input mediante `for` e `id`
- Elegir el tipo de input correcto en cada caso
- Todos los campos deben ser obligatorios

## Importante
No hace falta usar CSS ni JavaScript.

## Resultado esperado
El formulario debe verse ordenado, claro y listo para completar.
