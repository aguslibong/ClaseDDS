# Programa Analítico

Este programa analítico contempla los contenidos mínimos previstos en el DC vigente, así como aquellos considerados necesarios para desarrollar los resultados de aprendizaje propuestos.

## Unidad 1: Iniciación al Desarrollo Full Stack

**Carga horaria:** 20 hs

### Contenidos

- **Stack tecnológico:** front-end (aplicación SPA con React) y back-end (aplicación de microservicios con Node.js y Express.js). JavaScript (Vanilla), ECMAScript y pruebas en JavaScript con Jest (https://jestjs.io/). Rol y responsabilidades del desarrollador full stack.
- **Arquitectura cliente-servidor:** arquitectura de microservicios vs. arquitectura monolítica.
- **Versionado de código:** trabajo colaborativo con GitLab. Configuraciones locales y globales. Comandos básicos, manejo de ramas, resolución de conflictos, buenas prácticas y uso de IA.

## Unidad 2: JavaScript

**Carga horaria:** 20 hs

### Contenidos

- **Sintaxis de JavaScript:** variables, expresiones, tipos de datos, operadores, funciones, estructuras de control, arrays, módulos y manejo de errores.
- **Objetos en JavaScript:** trabajo con objetos, modelo de objetos basado en prototipos, promesas e iteradores.
- **Desarrollo seguro en JavaScript.**

## Unidad 3: Desarrollo Web Back-End

**Carga horaria:** 34 hs

### Contenidos

- **Node.js:** paquetes, dependencias, librerías y frameworks. Desarrollo de servidores y gestión de eventos asíncronos.
- **ORM:** conceptos básicos de estructura de bases de datos: creación de bases de datos, tablas y operaciones de manipulación de datos. Mapeo objeto-relacional (ORM).
- **Arquitectura:** principios de arquitectura: acoplamiento, cohesión, modularidad y unicidad. Identificación de patrones.
- **REST:** protocolo HTTP y verbos. Uso de JSON. Conceptos REST. Estructura de URI y ubicación de recursos. Express, servidor web, creación de API CRUD, CORS, Swagger, Postman y curl.
- **Pruebas unitarias:** concepto, ventajas, construcción y uso de Jest.
- **Segurización de APIs:** alternativas y uso de JSON Web Token (JWT). Identificación de patrones.
- **Desarrollo seguro aplicado al back-end.**

## Unidad 4: Desarrollo Web Front-End

**Carga horaria:** 34 hs

### Contenidos

- **HTML y CSS:** introducción al front-end (HTML y CSS). Concepto de servidor estático vs. aplicación dinámica. Diferencia entre página y aplicación. Servidor de recursos vs. servidor de aplicación. SPA (Single Page Application).
- **DOM (Document Object Model):** diseño responsive para distintos dispositivos. Uso de Bootstrap.
- **React:** conceptos iniciales y estructura de aplicación. Renderizado con Virtual DOM (funcionamiento). Estructura de componentes. Componentes padres, principio de responsabilidad única en componentes hijos, alta cohesión y bajo acoplamiento. Componentes de clase vs. funcionales. Propiedades. Uso de funciones callback. Hooks iniciales (useState, useEffect). Ciclo de vida de componentes, JSX, formularios y ruteo.
- **Identificación de patrones.**
- **Desarrollo seguro aplicado al front-end.**

## Unidad 5: Integración Front-End y Back-End

**Carga horaria:** 20 hs

### Contenidos

- Integración del front-end con el back-end.
- Log de aplicaciones.
- Despliegue.
- Desarrollo seguro en la integración front-end y back-end.
